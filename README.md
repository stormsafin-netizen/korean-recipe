# 韓国料理レシピアプリ 🇰🇷

冷蔵庫の食材から作れる韓国料理を探せるWebアプリです。全60品のレシピを収録。
スマホ最適化済みで、ホーム画面に追加すればアプリのように使えます(PWA対応)。

## 機能

- **食材から探す** — 食材を入力 or よく使う食材タグをタップすると、作れる韓国料理をマッチ度順に表示
- **レシピ一覧・検索** — 料理名(日本語・韓国語)・カテゴリで検索してレシピを確認
- **レシピ詳細** — 材料(メイン・サブ・調味料)、作り方、コツをボトムシートで表示。下スワイプで閉じられます

## スマホ対応

- 下部ナビゲーション(タブバー)とタッチ操作に最適化したUI
- ノッチ・ホームバー対応(セーフエリア)、入力時の自動ズーム防止
- PWA対応: ホーム画面に追加でフルスクリーン起動、オフラインでも動作

## 使い方

`index.html` をブラウザで開くだけで動作します。

PWA機能(ホーム画面追加・オフライン)を使う場合はHTTPSでの配信が必要です:

```bash
# ローカルで試す場合
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## ファイル構成

| ファイル | 内容 |
|---|---|
| `index.html` | アプリ本体(HTML/CSS/JS・レシピデータ込み) |
| `manifest.webmanifest` | PWAマニフェスト |
| `sw.js` | Service Worker(オフラインキャッシュ) |
| `icon.svg` / `icon-maskable.svg` | アプリアイコン |
